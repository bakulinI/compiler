Тема: Разработка препроцессора

## Описание
Программа на Go реализует модуль препроцессора, который очищает исходный код от:
- однострочных комментариев
- многострочных комментариев
- лишних пробелов
- пустых строк

После предварительной обработки программа выполняет лексический анализ очищенного исходного кода. Лексический анализатор последовательно просматривает текст и разбивает его на лексемы следующих категорий:
- ключевые слова
- идентификаторы
- целочисленные константы
- вещественные константы
- строковые константы
- булевы константы
- операторы
- разделители

Для каждой найденной лексемы формируется токен с типом, значением, номером строки и позицией в строке. Также строится таблица уникальных лексем и фиксируются лексические ошибки, например неизвестные символы, незакрытые строки и некорректные числовые константы.

## Запуск
```bash
go run .
```
